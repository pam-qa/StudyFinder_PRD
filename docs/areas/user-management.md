# 4.10 User Management

[← Analytics](analytics.md) · [Home](../index.md)

User Management controls platform access and user-level permissions.

## Requirements

- Admins should be able to manage users, roles, and account access.
- User search should return accurate results.
- User lists should support alphabetical sorting.
- Role-based permissions should control access to actions such as editing, creating, or managing
  studies.
- Status changes should include confirmation prompts when appropriate.
- Permissions should be consistently applied across StudyFinder features.
- The number of users and invitation capabilities should follow the active
  [subscription tier](../03-subscription-tiers.md).

# 4.6 Email Sequences

[← Companies, Sponsors & Contacts](companies-sponsors-contacts.md) · [Home](../index.md) · Next: [Templates →](templates.md)

Email Sequences allow users to automate outreach to relevant contacts using reusable templates and
scheduled steps.

## Core functionality

- Create multi-step email sequences.
- Define delays between sequence steps.
- Use templates for reusable outreach content.
- Enroll contacts in sequences.
- Track sequence progress and replies.
- Stop or adjust automation when a recipient replies.

## Requirements

- Delay logic should be based on the previous sequence step.
- Reply detection should prevent unnecessary follow-up messages.
- Contacts added to a sequence should receive the appropriate starting step.
- Sequence entry dates and labels should update correctly when schedules change.
- Pagination should support more than 10 records reliably.
- Attachments should display correctly regardless of text size.
- Email composition should support signatures and logos.
- Users should have access to a live preview before activating a sequence.
- AI-assisted sequence drafting may be available on demand.
- Digest email functionality may summarize outreach activity.
- AI Email Outreach features should only be available to **Growth** and **Enterprise** users
  ([subscription tiers](../03-subscription-tiers.md)).

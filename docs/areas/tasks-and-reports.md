# Tasks and Reports

Tasks and Reports help users track outreach work and summarize study discovery activity.

## Requirements

- Users should be able to create and manage tasks related to studies, contacts, or outreach.
- Reports should support relevant date ranges and filtering.
- Business-day calculations should account for configured holidays where applicable.
- Reporting should make it easy to identify completed work, active opportunities, and pending
  follow-up.
- Advanced or custom reporting should be restricted to **Enterprise** users where applicable
  ([subscription tiers](../03-subscription-tiers.md)).

# StudyFinder — Product Requirements Document (Wiki)

**StudyFinder** helps patients, caregivers, and referring clinicians discover clinical trials and
research studies they may be eligible for — then understand eligibility and connect with the study
team. This repository is the living PRD, organized as a wiki of linked pages.

> **Status:** Draft · **Owner:** Product · **Last updated:** 2026-07-02

---

## 📖 How to use this wiki

Each page below is a self-contained section of the PRD. Start with **Vision & Scope** for the big
picture, then drill into the area you care about. Pages cross-link with `[[wiki-style]]` intent via
regular Markdown links so everything renders natively on GitHub.
